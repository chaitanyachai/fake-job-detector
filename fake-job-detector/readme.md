# Fake Job Detector

A machine learning-powered web app built with Flask that analyzes job descriptions to detect potential fake or scam postings.

## 🚀 Features
- Paste any job description and get instant analysis.
- Clear visual feedback (alerts for fake/genuine jobs).
- History page to track past analyses.
- Modern UI with Copilot-style design.

## 🛠️ Tech Stack
- Python, Flask
- Scikit-learn, Pandas
- HTML, CSS, JavaScript
